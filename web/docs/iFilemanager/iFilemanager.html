		paste (line 804)
	
 
	
	

Paste files into directory with specific options.


Options format:  
  • int "max_size"   - max size in bytes
  • bool "dry_run"   - do not throw exceptions
  • bool "skip_dirs" - make mirror of all dirs in filemanager or just skip them
  • bool "override"  - see iFilesystem->copy()
  • bool "dynamic_name" - see iFilesystem->copy()
  • array "image"       - see iFilesystem->copy()
  • array "thumbs"      - see iFilesystem->copy()
  • int   "chmod"       - set mode for processed files
  • bool  "move"        - if true then old files will be removed
  • array "allowed_types" - array of allowed file extensions
