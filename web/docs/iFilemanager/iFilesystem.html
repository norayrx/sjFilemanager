copy (line 400)

Copies a file.

By default, if the target already exists, it is not overriden. To override existing files, pass the "override" option. To create uniq file name, pass the "dynamic_name" option. To process images files, you may pass "image" and "thumbs" options like arrays. Image option format:

  • int "width" - width
  • int "height" - height
  • string "type" - resize mode (width, height, auto, zoom, crop)
  • string "crop" - crop type (left-top, left-bottom, right-top, right-bottom, center, specific array('x' => int, 'y' => int))
If you passed the "crop" image option then after resizing image will be croped using some of specific methods, otherwise it will be only resized

  • return: if file copied

readDir (line 202)

Read directory using built in iterators.

Options format:

  • int "flags" - use only for recursive mode. Passed in RecursiveIteratorIterator constructor
  • string "skip" - skip regular expression.
  • bool "relative" - return list of files with relative path. By default return absolute path for each file.
  • bool "with_dir" - use for relative mode to return file's path with current dir name
  • bool|callback "sort" - sort files by specific callaback. By default use $this->doSort method

  • return: list of files and/or directories
  • see: iFilesystem::doSort()
  • access: public

  • string $path: path to directory
  • string $mode: if mode equal "r" is recursive mode, otherwise use simple mode
